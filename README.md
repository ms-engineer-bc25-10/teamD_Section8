# 推ししか勝たん貯金（Oshi Savings）

推し活は、
騒いでも、静かでも、語ってもいい。

このアプリは、
そのすべてを「等価な入金」として扱います。

## 概要

「推し活という行動」を、銀行APIを使って **“入金体験”として可視化**する学習用ミニアプリです。  
ボタンを押すだけで、推し活口座の残高確認や、親口座 → 推し活用つかいわけ口座への振替（入金）を体験できます。

- 目的：Fintech / 銀行API / Embedded Finance の構造を **最小実装で体感する**
- DBなし（状態は銀行APIレスポンスとフロント表示のみ）
- ローカル実行前提（デプロイなし）

---

## デモ体験（MVP）

1. 残高確認ボタンで「推し活用つかいわけ口座」の残高を表示
2. 推し活ボタンで、親口座 → 推し活口座へ振替（入金）
3. 明細表示で、入出金履歴を確認
4. （特殊）「全額献上」ボタンで、推し活口座からの全額送金フローを疑似体験

---

## アーキテクチャ（重要）

フロントエンドから銀行APIを直接呼ばず、**必ず Next.js Route Handlers（/app/api）を経由**します。
